# 术语表 / Glossary

| 中文 | English | 定义 |
|---|---|---|
| 逻辑资产 | Asset | 用户认知中的同一内容 |
| 来源对象 | SourceObject | 数据源中的文件/对象 |
| 内容版本 | AssetVersion | 内容变化后的版本 |
| 媒体表现 | Rendition | 编码、分辨率、字幕等表现 |
| 物理副本 | Replica | 实际存储对象 |
| 衍生资产 | DerivedArtifact | AI、OCR、ASR 等结果 |
| 完整缓存 | Complete Cache | 可淘汰的完整文件 |
| 正式副本 | Formal Replica | 验证且受保留策略管理 |
| 保留固定 | Retention Hold | 阻止普通淘汰的固定 |
| 回源 | Origin Fetch | 从来源读取内容 |
| 安全水位 | Safety Floor | 必须保留的最小可用空间 |
| 规则链 | Chain | LiteFlow 可执行编排 |
| 节点 | Node | LiteFlow 同步组件 |
| 长任务 | Workflow | Temporal 可恢复执行 |
| 证据 | Evidence | 可复现测试和运行材料 |
